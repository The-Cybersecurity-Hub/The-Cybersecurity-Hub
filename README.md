🔐 Cybersecurity Hub

A free, open-source cybersecurity tool directory covering 30+ domains and 90+ tools — hosted on GitHub Pages.


🚀 Deploy to GitHub Pages (3 Steps)

Step 1 — Create a GitHub Repository


Go to github.com → Sign in or create a free account

Click "New repository" (green button top right)

Name it: cybersecurity-hub

Set it to Public

Click "Create repository"


Step 2 — Upload the Files

Option A – Drag & Drop (easiest):



Open your new repository on GitHub

Click "uploading an existing file"

Drag the entire cybersecurity-hub folder contents into the browser

Click "Commit changes"


Option B – GitHub Desktop:



Download GitHub Desktop

Clone your new repo locally

Copy all files from this folder into the cloned folder

Commit and Push


Option C – Git CLI:


cd cybersecurity-hub
git init
git add .
git commit -m "Initial commit – Cybersecurity Hub"
git remote add origin https://github.com/YOUR_USERNAME/cybersecurity-hub.git
git push -u origin main

Step 3 — Enable GitHub Pages


Go to your repository → Settings tab

Scroll down to Pages (left sidebar)

Under Source → select "Deploy from a branch"

Branch: main / Folder: / (root)

Click Save


✅ Your site will be live in ~2 minutes at:
https://YOUR_USERNAME.github.io/cybersecurity-hub
